# O que é?

Projeto baseado no conteúdo fornecido 
na IMERSÃO JAVA pela ALURA.

# Tecnologias

- java openjdk 11.0.15

# Gerenciador de SDK

- [sdkman](https://sdkman.io/)

# Build

```bash
javac -Xlint -d bin/ -cp src/ src/App.java
```

# Run
```bash
bin$ java App
```
